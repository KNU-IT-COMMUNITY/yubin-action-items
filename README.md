날짜별 투두 리스트를 관리하는 작은 웹입니다!<br />
저는 노트 다이어리를 쓸 때 할 일들을 마구 적어놓고 시간대 별로 분배를 하는데요, 이러한 Todo-List를 웹으로 구현하면 좋을 것 같다고 생각했습니다.<br />
메인 페이지에서 오늘 투두를 확인하고, 캘린더에서 날짜를 선택하면 해당 날짜의 투두를 작성할 수 있습니다!

## 요구사항 checklist
(필수) 메인 페이지에서 투두리스트 페이지로 이동하는 형태로 페이지를 구성해주세요.✅<br />
(필수) 오늘의 투두리스트를 작성, 수정, 삭제, 조회할 수 있도록 만들어주세요.✅<br />
작성 시에는 단순 투두리스트의 내용만 넣는게 아니라, 날짜도 같이 넣을 수 있도록 만들어주세요.✅<br />
(선택) 서비스에서 다크모드와 라이트모드를 지원할 수 있도록 만들어주세요.❎ <br />
(선택) 날짜순으로 정렬이 될 수 있도록 정렬 기능을 만들어주세요.✅<br />

## 주요 기능
- 메인 페이지: 오늘 투두 리스트 요약 표시
- 캘린더 날짜 선택 → 해당 날짜의 Todo-List를 입력할 수 있는 페이지로 이동
- 투두 페이지: 투두 플로팅 생성 + 타임라인 드래그 배치
- 투두 수정/삭제(hover 시 버튼 표시)
- localStorage에 날짜별로 저장

## 기술 스택
- React + TypeScript + Vite
- Emotion (css prop)
